# DCC 1.0.1 Audit Correctness Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Make every detected finding and audit row join through a deterministic, exact finding identity, reject invalid execution plans before data changes, and certify every supported input format without chunking warnings or skipped tests.

**Architecture:** `R/findings.R` owns additive, deterministic finding IDs. `R/execute.R` validates the complete execution request, propagates IDs into audit rows, and records unmatched findings as unhandled. `R/report.R` reconciles only on the exact ID and turns integrity violations into typed errors. `R/chunked.R` owns CSV/TSV schema locking and Arrow streaming; format-matrix tests exercise every `dcc_read()` backend with fixed or generated fixtures. Existing public columns and valid action calls remain compatible.

**Tech Stack:** R (>= 4.1), data.table, testthat edition 3, arrow, haven, readxl, writexl, jsonlite, hand-maintained Rd documentation.

```r
new_finding_ids <- function(run_id, check_id, record_id, variable) {
  record_part <- ifelse(is.na(record_id), "<group>", record_id)
  variable_part <- ifelse(is.na(variable), "<record>", variable)
  key <- paste(run_id, check_id, record_part, variable_part, sep = "\r")
  occurrence <- ave(seq_along(key), key, FUN = seq_along)
  paste(run_id, check_id, record_part, variable_part, occurrence, sep = "|")
}

dcc_findings <- function(record_id = character(), variable = NA_character_,
                         check_id = character(), evidence = character(),
                         severity = "warn", dimension = NA_character_,
                         run_id = "manual") {
  n <- max(length(record_id), length(evidence))
  if (n == 0L) return(empty_findings())
  if (length(run_id) != 1L || is.na(run_id) || !nzchar(run_id)) {
    dcc_abort("`run_id` must be one non-empty string.", class = "dcc_type_error")
  }
  severity <- as.character(severity)
  bad <- setdiff(unique(severity), c("info", "warn", "fail"))
  if (length(bad)) {
    dcc_abort("Invalid severity value(s): ", paste(bad, collapse = ", "),
              ". Use \"info\", \"warn\" or \"fail\".", class = "dcc_type_error")
  }
  record_id <- rep_len(as.character(record_id), n)
  variable <- rep_len(as.character(variable), n)
  check_id <- rep_len(as.character(check_id), n)
  out <- data.table::data.table(
    finding_id = new_finding_ids(as.character(run_id), check_id, record_id, variable),
    record_id = record_id, variable = variable, check_id = check_id,
    evidence = rep_len(as.character(evidence), n),
    severity = rep_len(severity, n), dimension = rep_len(as.character(dimension), n)
  )
  data.table::setattr(out, "class", c("dcc_findings", class(data.table::data.table())))
  out
}
```

Validate that `run_id` is exactly one non-missing, non-empty string with
`dcc_type_error`. Make `empty_findings()` include
`finding_id = character()`; retain it in `bind_findings()`.

```r
f <- data.table::copy(data.table::as.data.table(x$findings))
f[, `:=`(action = NA_character_, status = "unhandled", handled = FALSE)]
if (nrow(log) && any(!log$finding_id %in% f$finding_id)) {
  dcc_abort("Audit log contains unknown finding_id.", class = "dcc_reconcile_error")

per_finding <- log[, .(actions = list(unique(action))), by = finding_id]
```
